# Prototipo Delivery App - Variante B

Una aplicación web de delivery de comida con interfaz moderna y funcionalidad completa.

## 🚀 Características

- **Selección de Restaurantes**: Explora diferentes categorías de comida
- **Menú Interactivo**: Agrega productos al carrito con contador en tiempo real
- **Carrito de Compras**: Visualiza y gestiona tu pedido
- **Navegación Fluida**: Entre restaurantes, menú y confirmación
- **Diseño Responsivo**: Optimizado para móvil y desktop

## 🎨 Tema Personalizado

- **Header**: Azul oscuro (#1e40af)
- **Acento**: Naranja (#f97316)
- **Fondo**: Blanco limpio
- **Sin navbar**: Interfaz simplificada sin indicadores de progreso

## 🛠️ Tecnologías

- **HTML5**: Estructura semántica
- **CSS3**: Variables CSS, Flexbox, Grid
- **Vanilla JavaScript**: Funcionalidad interactiva
- **Git**: Control de versiones

## 🚀 Cómo Ejecutar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/GeorgieRe2rn/deliveryfoodB.git
   cd prototipo-delivery-b
   ```

2. Abre `index.html` en tu navegador o usa un servidor local:
   ```bash
   python -m http.server 8000
   ```

3. Visita `http://localhost:8000`
